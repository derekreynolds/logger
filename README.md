# logger
A Node.js module that to wrap Winston logger
## Installation 
```sh
npm install @derekreynolds/logger --save
yarn add @derekreynolds/logger
bower install @derekreynolds/logger --save
```

```
### TypeScript
```typescript
import * as logger from 'logger';
logger.info(message)
```

## Test 
```sh
npm run test
```
